# 🔍 Lost and Found Management System

A desktop application built with **Java** and **JavaFX** that helps users report and discover lost or found items. The system provides a secure platform where users can create accounts, log in, and manage lost-and-found records through an intuitive graphical interface.

## ✨ Features

* 🔐 User Registration and Login
* 👤 Secure User Authentication
* 📢 Post Lost Items
* 📦 Post Found Items
* 🔎 Browse Available Lost & Found Reports
* 💾 File-Based Data Storage (No Database Required)
* 🖥️ User-Friendly JavaFX Interface

## 🛠️ Technologies Used

* **Java**
* **JavaFX**
* **File System Storage**
* **Object-Oriented Programming (OOP)**

## 🚀 How It Works

1. Create an account or log in with existing credentials.
2. View lost and found items posted by other users.
3. Post details about a lost item.
4. Report an item that has been found.
5. Search and browse records to help reunite owners with their belongings.
